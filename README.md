# React Static Template

**Requirements**

```
node: ^7.0.0
npm: ^4.0.0
```

**Install**

```
// yarn
yarn

// npm
npm i
```

**Use**

```
// only for development, server on localhost:3000
// yarn
yarn start

// npm
npm run start
```
```
// only for production, server on localhost:8080
// yarn
yarn prod

// npm
npm run prod
```
```
// only for production, read-only, build
// yarn
yarn build

// npm
npm run build
```
```
// only for production, read-only, uglified build
// yarn
yarn build:min

// npm
npm run build:min
```
